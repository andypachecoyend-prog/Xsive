Guía de Uso

Cómo funciona el motor de curaduría espectral de Xsive.

01

Instalar dependencias

pip install flask flask-cors librosa
pip install tensorflow scikit-learn
pip install numpy opencv-python
pip install matplotlib seaborn

Requiere Python 3.9+ y opcionalmente FFmpeg para MP3.

02

Preparar el dataset

SIVE/
├── data/
│   ├── vanguardista/  ← 1000 tracks (Clase 1)
│   └── antiguo/       ← 1000 tracks (Clase 0)
└── xsive_classifier/  ← Código ML

Formatos: WAV, MP3, FLAC, AIF, OGG, M4A.

03

Iniciar el servidor

cd SIVE
python server.py

# Interfaz web:
# http://localhost:5000

El modelo se carga automáticamente si ya existe.

04

Entrenar desde CLI

# Pipeline completo
python xsive_classifier/main.py

# Solo inferencia
python xsive_classifier/main.py \
  --predict track.wav

# Reentrenar (sin reextracción)
python xsive_classifier/main.py \
  --from-processed
